# Groq Chatbot

A simple chatbot using Groq Cloud LLM API.

## Setup

1. Sign up for Groq Cloud at https://console.groq.com/
2. Generate an API key from your dashboard.
3. Replace `your_groq_api_key_here` in the `.env` file with your actual API key.

## Installation

Install the required packages:

```bash
pip install -r requirements.txt
```

## Running the Chatbot

Run the chatbot:

```bash
python chatbot.py
```

Type your messages and press Enter. Type 'exit' to quit.
